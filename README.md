## What is Headless Shopify?

Headless Shopify architecture decouples the frontend (what customers see) from the backend (Shopify's commerce engine). This allows brands to:

- **Achieve faster load times** - Custom frontends optimized for performance
- **Full design freedom** - No theme limitations
- **Omnichannel experiences** - Same backend, multiple frontends
- **Advanced personalization** - Tailored shopping experiences
- **Global scalability** - Handle high traffic with custom infrastructure

Common frontend frameworks used:
- [Shopify Hydrogen](https://hydrogen.shopify.dev/) - Shopify's official React framework
- [Next.js](https://nextjs.org/)
- [Gatsby](https://www.gatsbyjs.com/)
- [Nuxt.js](https://nuxtjs.org/)
- [Remix](https://remix.run/)

## Tech Stack Examples

Common technology combinations used in headless Shopify implementations:

### Shopify Hydrogen + Oxygen
```
Frontend: Shopify Hydrogen (React-based)
Hosting: Shopify Oxygen
Backend: Shopify Storefront API
```

### Next.js + Vercel
```
Frontend: Next.js
Hosting: Vercel
Backend: Shopify Storefront API
CMS: Sanity / Contentful (optional)
```

### Gatsby + Netlify
```
Frontend: Gatsby
Hosting: Netlify
Backend: Shopify Storefront API
```

### Remix + Cloudflare
```
Frontend: Remix
Hosting: Cloudflare Workers
Backend: Shopify Storefront API
```

## Contributing

Contributions are welcome! Please read the contribution guidelines first.

### How to Contribute

1. Fork this repository
2. Create a new branch: `git checkout -b add-new-store`
3. Add your store to the appropriate category
4. Commit your changes: `git commit -am 'Add StoreName'`
5. Push to the branch: `git push origin add-new-store`
6. Submit a pull request

### Contribution Guidelines

- **Verify the store is headless** - Should use Shopify as backend with a custom frontend
- **Include accurate information** - Store name, description, and working URL
- **Follow the format** - Match the existing table structure
- **One store per PR** - Makes review easier
- **No affiliate links** - Keep links clean and direct
